Optimized KH Conditioning Project

Develop a software tool for ranking multiple geological model realizations with robust KH values along the well bore. This tool creates multiple scenarios for different values of KH, creates the necessary GIGA POWERS models for each run and the allows the user to compare and rank resulting simulation values when compared with actual field data.

The first version of the software is already available for the users.
